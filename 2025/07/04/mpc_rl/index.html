# 🚗 比较 MPC 与 强化学习：以小车到目标点为例

本例以一个简单的小车控制任务，展示 MPC（Model Predictive Control） 与 强化学习（Reinforcement Learning） 的形式对比与数学关系

我们考虑一维小车的位置控制问题。状态为 xtx_txt​（当前位置），动作为加速度 utu_tut​。目标是从起点 x0x_0x0​ 移动至目标位置 xgoalx_{\text{goal}}xgoal​。

# 系统动力学模型（离散时间）

xt+1=f(xt,ut)=xt+ut⋅Δtx_{t+1} = f(x_t, u_t) = x_t + u_t \cdot \Delta txt+1​=f(xt​,ut​)=xt​+ut​⋅Δt

# 代价函数（Cost Function）

c(xt,ut)=(xt−xgoal)2+λut2c(x_t, u_t) = (x_t - x_{\text{goal}})^2 + \lambda u_t^2c(xt​,ut​)=(xt​−xgoal​)2+λut2​

控制目标：寻找一系列控制量 {u0,u1,…}\{u_0, u_1, \dots\}{u0​,u1​,…}，使得总代价最小。

对于 MPC 而言，每一时刻都解一个有限时域优化问题：

当前时刻为 ttt，优化未来 HHH 步控制序列：

min⁡ut:t+H−1∑k=0H−1[(xt+k−xgoal)2+λut+k2]\min_{u_{t:t+H-1}} \sum_{k=0}^{H-1} \left[(x_{t+k} - x_{\text{goal}})^2 + \lambda u_{t+k}^2\right]ut:t+H−1​min​k=0∑H−1​[(xt+k​−xgoal​)2+λut+k2​]

受限于：

xt+k+1=xt+k+ut+k⋅Δtx_{t+k+1} = x_{t+k} + u_{t+k} \cdot \Delta txt+k+1​=xt+k​+ut+k​⋅Δt

  • 解出控制序列 {ut∗,ut+1∗,…}\{u_t^*, u_{t+1}^*, \dots\}{ut∗​,ut+1∗​,…}
  • 仅执行第一个动作 ut∗u_t^*ut∗​
  • 下一步重新优化

# 强化学习（以 Q-Learning 为例）

强化学习目标是最大化累计奖励（即最小化总代价）。

将代价函数转化为负的即时奖励：

r(st,at)=−[(xt−xgoal)2+λat2]r(s_t, a_t) = -[(x_t - x_{\text{goal}})^2 + \lambda a_t^2]r(st​,at​)=−[(xt​−xgoal​)2+λat2​]

强化学习的目标是学习一个策略 π(a∣s)\pi(a \mid s)π(a∣s)，使得以下期望累计奖励最大，：

max⁡πEπ[∑t=0∞γtr(st,at)]\max_\pi \mathbb{E}_\pi \left[ \sum_{t=0}^{\infty} \gamma^t r(s_t, a_t) \right]πmax​Eπ​[t=0∑∞​γtr(st​,at​)]

等价于最小化 MPC 中的累积代价：

min⁡πEπ[∑t=0∞γtc(xt,ut)]\min_\pi \mathbb{E}_\pi \left[ \sum_{t=0}^{\infty} \gamma^t c(x_t, u_t) \right]πmin​Eπ​[t=0∑∞​γtc(xt​,ut​)]

# Q-Learning 原理推导

Q-learning 是一种无模型的强化学习算法，通过学习一个状态 - 动作值函数 Q(s,a)Q(s, a)Q(s,a)，表示在状态 sss 选择动作 aaa 后，未来获得的期望总奖励。

# Bellman 方程定义：

Q(st,at)=r(st,at)+γmax⁡a′Q(st+1,a′)Q(s_t, a_t) = r(s_t, a_t) + \gamma \max_{a'} Q(s_{t+1}, a')Q(st​,at​)=r(st​,at​)+γa′max​Q(st+1​,a′)

即：当前的 Q 值应等于当前奖励 + 未来最大奖励的折扣。

# Q-learning 更新公式：

在交互过程中迭代更新：

Q(st,at)←Q(st,at)+α[r(st,at)+γmax⁡a′Q(st+1,a′)−Q(st,at)]Q(s_t, a_t) \leftarrow Q(s_t, a_t) + \alpha \left[ r(s_t, a_t) + \gamma \max_{a'} Q(s_{t+1}, a') - Q(s_t, a_t) \right]Q(st​,at​)←Q(st​,at​)+α[r(st​,at​)+γa′max​Q(st+1​,a′)−Q(st​,at​)]

  • α\alphaα 是学习率
  • γ\gammaγ 是折扣因子
  • r(st,at)r(s_t, a_t)r(st​,at​) 是立即奖励
  • Q(st+1,a′)Q(s_{t+1}, a')Q(st+1​,a′) 是未来的最大收益

最终收敛后策略为：

π(s)=arg⁡max⁡aQ(s,a)\pi(s) = \arg\max_a Q(s, a)π(s)=argamax​Q(s,a)
